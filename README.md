# Divine Earthly Intelligence (DEI)

## Vision
Divine Earthly Intelligence is a multi-agent framework designed to address global challenges in **Agriculture, Climate Risk, and Disaster Prediction**. By combining symbolic reasoning with statistical AI, DEI provides robust, explainable, and scalable intelligence for critical earthly systems.

## Core Architecture
- **Hybrid Intelligence Engine**: Integrates rule-based symbolic logic (for high-risk triggers) and statistical model inference (for general predictions).
- **Multi-Agent System (MAS)**: A collaborative network of specialized agents:
  - **Planner**: Decomposes complex user queries into actionable steps.
  - **Research**: Gathers and simulates data retrieval.
  - **Critic**: Evaluates outputs for quality and accuracy.
  - **Optimizer**: Refines results based on critical feedback.
- **Domain Modules**: Specialized logic for Agriculture (crop health), Climate Risk (trend analysis), and Disaster Prediction (threat identification).

## How to Run
### Streamlit Dashboard
To launch the interactive dashboard:
```bash
streamlit run demo/app.py
```

### Colab Demonstration
Open `colab/demo_script.py` in a Google Colab environment and execute to see the end-to-end workflow including the self-improvement loop.

## Environment Optimization
This platform is optimized for **CPU-only and free-tier environments** (like Google Colab or local laptops). It does not require a GPU to perform inference or agent orchestration.
